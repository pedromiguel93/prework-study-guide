HTML

  • The head element contains information about the webpage.
  • The title element is very useful for SEO purposes.
  • The body element represents the visible content shown to the user.
  • h1, h2, h3, h4, h5, h6 headers should be used in order of importance.
  • ul and ol elements are used to created unordered and ordered lists respectively.
  • li elements are used to list specific items within a list.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A period preceding some text designates is at a class selector.
  • The border property assigns a border to an element.
  • The box-shadow CSS property adds shadow effects around an element's frame.
    A box shadow is described by X and Y offsets relative to the element, blur and spread radius, and color.

Git

  • git status: checks what branch we are currently on.
  • git branch: checks existing branches.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git add -A: moves changes to staging area.
  • git commit -m "add your commit message": takes changes in staging area and adds them to working branch with a message.
  • git branch -d branch-name: deletes local branch.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • A function is a collection of code to be run when the function is called.
  • An if statement will only execute a set of code when specific conditions are met.
  • An array is used to store a group of related data.
  • A for Loop runs a set of code until a specific condition is met.
  • Control flow is the order in which a computer executes code in a script.
